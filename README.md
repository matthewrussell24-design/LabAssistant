# LabAssistant

LabAssistant is a beginner-friendly Streamlit application for exploring laboratory data.

## Version 0.1 Goal

This first milestone keeps the app intentionally small:

- Launch locally
- Show a title
- Upload a CSV file
- Display the uploaded data in a table

## Local Setup

Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
