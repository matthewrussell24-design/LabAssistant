# LabAssistant Ideas

This is the repository inbox for promising ideas that are worth preserving but
are not yet approved, sequenced, or detailed enough to become roadmap work.

## How to Use This File

- Capture each idea briefly, including the problem or opportunity it addresses.
- Record useful context, constraints, or open questions when known.
- Do not treat an entry here as a commitment or active task.
- Move an idea to `ROADMAP.md` when it is prioritized and sequenced.
- Create a numbered file in `prompts/` when implementation is ready to begin.
- Record durable architectural decisions in `decisions/`, after a decision has
  actually been made.
